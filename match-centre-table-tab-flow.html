Table Tab Flow

The Table tab displays the overall league standings for the United Rugby Championship.

Table Overview

  • The Table screen shows all teams ranked based on their current league position.

  • Teams are ordered according to competition rules and points.

Standings Table Layout

The standings table is displayed as a vertically scrollable list where each row represents a team.

Each row contains:

  • Rank – Current league position

  • Team – Team logo and team name

  • P – Matches played

  • W – Matches won

  • D – Matches drawn

  • L – Matches lost

  • PD – Points difference

  • PTS – Total league points

User Flow

Match Centre → Table Tab → View League Standings

  • The user can scroll vertically to view all teams in the league table.

  • No navigation is triggered when a team row is selected (read-only informational view).

Season Context Behavior

  • The standings table reflects the currently selected season from the top bar.

  • When the season is changed, the table updates to show standings for that season.
