App Environments

The app supports two runtime environments mapped directly to build variants.

1. Production (Prod)

  • Used for live users via the Play Store.

  • GraphQL Endpoint:
    https://unitedrugby.com/graphql

2. Staging (Dev)

  • Used for internal testing and development.

  • GraphQL Endpoint:
    https://staging.unitedrugby.com/graphql

3. Build Variant Mapping

  • prod variant

    • Uses Production GraphQL endpoint:
      https://unitedrugby.com/graphql

  • dev variant

    • Uses Staging GraphQL endpoint:
      https://staging.unitedrugby.com/graphql
