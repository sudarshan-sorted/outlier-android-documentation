Play Screen Flow

The Play screen provides access to interactive games and competitions.

1. Top Bar

  • Sticky top bar is present as in Home.

  • Center title changes to "Play".

  • Left and right icons remain consistent with Home behavior.

2. Entry Point

  • User navigates to the Play tab from the main navigation menu.

3. Content Layout

  • Displays a vertical list of cards, each representing a different interactive experience, such as:

    • Predictor

    • Polls

    • Fantasy

    • Competitions

    • Other game-related features

4. User Interactions

  • Clicking on a card: Redirects the user to an external website in a web view or the default browser.

  • The app does not host these games internally; it acts as a gateway to the external platform.

5. Summary Flow

Main Screen → Play Tab → Select Card → External Website
