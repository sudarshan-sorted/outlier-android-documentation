Players Tab Flow

The Players tab displays the squad list for the selected team, with a player filter and player profile cards.

1. Default State

  • When the Players tab is opened:

    • The Forwards section is displayed.

    • A list of player cards is shown in a vertical scrollable layout.

2. Player Filter

  • A player filter dropdown is shown on the right (e.g., All Players).

  • The filter allows selecting:

    • All Players

    • A specific individual player

Interaction

  • Tapping the dropdown opens a list of players.

  • Selecting a player updates the screen to show only that player's card.

  • Selecting All Players shows the full squad list again.

3. Player Card Layout

Each player is displayed as a large visual card containing:

  • Player image

  • Player name

  • Position (e.g., Lock, Flanker)

  • Age

  • Birthday

  • Height

  • Weight

Cards are stacked vertically when multiple players are shown.

4. Player List Interaction

  • User can scroll vertically to browse all players (when All Players is selected).

  • Tapping a player card opens the Player Detail screen.

5. User Flow Summary

  1. User opens Team Details → Players Tab.

  2. Forwards group and player list are shown by default.

  3. User opens the player filter dropdown.

  4. User selects a specific player or All Players.

  5. User scrolls or taps a player card to view detailed information.
