Overview Tab Flow

The Overview tab provides a high-level summary of the selected team, including basic team information, upcoming fixtures, highlights, and recent news.

1. Team Information Section

This section displays static metadata about the selected team.

  • Home Ground – The stadium where the team plays home matches.

  • Country – The team’s country.

  • Founded – The year the team was established.

This section is informational and does not contain interactions.

2. Upcoming Matches Section

  • Displays a horizontal pager showing upcoming matches for the selected team.

  • Each page contains a match card with:

    • Opponent team

    • Match date and time

    • Venue

    • Other match related data

Interaction with Upcoming Matches Section

  • Swiping horizontally navigates between upcoming matches.

  • Tapping a match card opens the Match Details screen.

3. Highlights Section

  • Displays Instagram-like story circles showing highlight stories for the selected team’s matches.

  • This component behaves the same as the Highlights component used in Home, Videos, and My Team sections.

Interaction with Highlights Section

  • Tapping a highlight opens the Stories Viewer to watch the team’s match highlights.

4. News Section

  • Displays a list of recent news articles related to the selected team.

  • Articles are shown in a vertical list of news cards.

Interaction with News Section

  • Tapping a news card opens the Article Detail screen.

  • View All button opens the main News screen showing all news articles.

5. User Flow Summary

  1. User opens Team Details → Overview Tab.

  2. User views team metadata (Home Ground, Country, Founded).

  3. User swipes through upcoming matches and taps a match for details.

  4. User taps highlights to watch team stories.

  5. User browses team-related news or taps View All to open the News screen.
