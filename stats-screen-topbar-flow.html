Top Bar Flow

The Stats screen uses a dedicated top bar configuration that remains consistent across all Stats subsections.

Top Bar Configuration

  • Left Icon (Menu Button)
    Opens the side navigation drawer, same behavior as Home and Match Centre.

  • Center Title
    Displays “Stats” and remains constant across all Stats sub-screens.

  • Right Action (Season Selector Dropdown)
    Allows the user to change the season context (e.g., 2025–26, 2024–25, 2023–24).

    Selecting a season updates all statistics shown within the Stats section, including:

    • Team statistics

    • Individual player statistics

    The entire Stats section refreshes to reflect the selected season’s data.
