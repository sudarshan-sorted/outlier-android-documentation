Broadcasters Flow

Broadcaster information is displayed on match cards across multiple surfaces in the application, including the Home screen, Match Centre, and other match-related views. This allows users to quickly identify where a match will be broadcast before it starts and during live matches.

The broadcaster list is resolved based on the user's detected region when available.

Broadcaster Resolution Flow

1. Region Detected

  • The app detects the user’s country using location services.

  • The match card displays broadcasters available in that country.

  • Some broadcasters may be global, while others are region-specific.

  • This helps users identify the correct platform or channel to watch the match in their region.

2. Region Not Detected

  • If the user’s country cannot be detected, the app displays all globally available broadcasters.

  • Broadcasters are shown without regional filtering.

  • This ensures broadcast information is still available in all scenarios.

User Flow

  • User opens Home, Match Centre, or another match-related surface.

  • User views a match card for an upcoming or live match.

  • The app resolves broadcasters based on detected region when available.

  • Broadcaster logos and names are displayed on the match card.

  • User uses this information to decide where to watch the match.

User Interaction Flow Summary

  • User opens match-related screens containing match cards.

  • User views broadcaster information on upcoming or live match cards.

  • System resolves broadcasters based on region when available.

  • System falls back to global broadcasters when region is unavailable.

  • User decides where to watch the match based on the displayed broadcasters.
