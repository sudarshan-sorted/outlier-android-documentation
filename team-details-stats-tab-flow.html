Stats Tab Flow

The Stats tab displays aggregated performance statistics for the selected team. Users can view season-specific team metrics grouped into expandable statistical categories.

1. Default State

  • The Attack category is expanded by default when the screen loads.

  • All other categories are collapsed initially.

2. Season Selector

  • A headline labeled Team is shown at the top of the stats section.

  • A season dropdown (e.g., 2025–26) allows users to switch between seasons.

  • Changing the season updates all team statistics displayed on the screen.

3. Stats Categories

Team statistics are grouped into collapsible sections:

  • Attack

  • Defence

  • Kicking

  • Discipline

  • Lineouts

  • Scrums

4. Category Interaction

  • Each category has a chevron toggle for expand/collapse.

  • Expanding a category reveals its metrics.

  • Collapsing a category hides its metrics.

5. Metrics Display

  • Each expanded category displays multiple stat metrics (label + numeric value).

  • Example metrics:

    • Points Scored

    • Tries Scored

    • Offloads

    • Metres Gained

    • Clean Breaks

    • Defenders Beaten

  • A ranking indicator (e.g., 6th) is shown for overall category ranking.

6. User Flow Summary

  1. User opens Team → Stats tab.

  2. Attack category is expanded by default.

  3. User selects a different season (optional).

  4. User expands or collapses stat categories.

  5. User reviews team performance metrics.
