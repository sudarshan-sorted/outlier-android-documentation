Tab Selector Flow

The Match Centre screen provides a tab-based navigation selector that allows users to switch between different match-related views.

Tab Options

The tab selector contains three primary tabs:

  • Rounds
    Displays match rounds and the list of fixtures or results within each round.

  • Table
    Displays the league standings table, showing team rankings and performance metrics (e.g., played, wins, draws, losses, points difference, and total points).

  • Shields
    Displays competition-specific shield standings, segmented by regions or groups (e.g., Irish, Scottish/Italian, South African, Welsh).

Interaction Flow

  1. The user lands on the Rounds tab by default.

  2. Tapping on Table switches the view to the league standings table.

  3. Tapping on Shields switches the view to shield-specific standings.

  4. The selected tab is visually highlighted to indicate the active section.

  5. Content below the tab selector updates dynamically based on the selected tab.

Behavior with Season Selector

  • The tab content is scoped to the currently selected season.

  • When the season is changed from the top bar, all tab views (Rounds, Table, Shields) refresh to reflect the selected season’s data.
