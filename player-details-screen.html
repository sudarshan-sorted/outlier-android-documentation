Player Details Screen Flow

The Player Details Screen provides comprehensive information and performance data for an individual player.

1. Navigation Entry

  • User navigates to this screen via:

    • Teams → [Select Team] → Players Tab → Select Player Card

  • Selecting a player opens the Player Stats Screen for that specific player.

2. Season Stats Section

This section displays the selected player’s current season performance summary.

Metrics Shown

  • Matches Played

  • Matches Started

  • Tries Scored

  • Total Points

3. Performance Categories Section

Below Season Stats, detailed performance metrics are grouped into expandable categories.

Default State

  • Attack category is expanded by default.

  • All other categories are collapsed initially.

Categories

Each category is collapsible/expandable:

  • Attack (Default Expanded)

  • Defence

  • Kicking

  • Discipline

  • Lineouts

4. Interaction Behavior

  • User can expand/collapse each category independently.

  • Metrics are displayed as label-value pairs, typically aligned in a list format.

5. Data Behavior

  • All metrics are season-scoped and tied to the selected player.

  • This screen reflects the currently selected season context.
