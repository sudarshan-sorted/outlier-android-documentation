Common Screen Flow

The Team Details screen provides information, fixtures, news, and videos for a selected team. This screen is accessible for any team selected from the Teams list in the side drawer.

1. Entry Point

Navigation Path:
Side Drawer → Teams → Select Team

  • User opens the side drawer.

  • Taps on Teams to expand the list of all URC teams.

  • Selecting a team opens the Team Details screen for that team.

2. Top Bar

The Team Details screen contains a persistent top bar.

  • Left: Back button

    • Navigates back to the previous screen.

  • Center: Selected Team Name

    • Displays the currently selected team’s name.

  • Right: Team Selector Dropdown

    • Allows the user to switch to another team without leaving the screen.

    • Shows a dropdown list of all teams.

3. Team Header Card (Common for All Tabs)

Below the top bar, a Team Header Card is displayed. This section remains the same across all tabs.

Team Information

  • Team logo

  • Team name

Action Buttons

  • Club Site

    • Opens the official team website in a web view or browser.

  • Buy Tickets

    • Redirects the user to the ticket purchasing page for the selected team.

  • Sync Fixtures

    • Provides an option to sync team fixtures with the user’s calendar (exact behavior depends on platform integration).

4. Tab Selector (Sub-Screens)

Below the Team Header Card, a tab selector is shown:

  • News

  • Fixtures

  • Videos

Each tab opens a corresponding sub-screen while keeping the top bar and team header card unchanged.
