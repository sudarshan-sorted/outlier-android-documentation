Tab Selector

The Stats screen contains a tab selector that allows users to switch between different statistical views.

1. Team Tab

  • Default selected tab when the Stats screen is opened.

  • Displays team-level aggregated statistics across the competition.

  • Each team is listed in a ranked order based on the selected metric category.

  • Stats are grouped into collapsible metric sections (e.g., Attack, Defence, Discipline, Set Pieces).

  • Each team row can expand to reveal detailed metrics such as:

    • Points scored

    • Tries scored

    • Offloads

    • Metres gained

    • Clean breaks

    • Defenders beaten

    • (and other relevant team performance metrics)

2. Players Tab

  • Displays individual player statistics across the competition.

  • Players are ranked based on the selected metric category.

  • Similar metric groupings are used (Attack, Defence, Kicking, Discipline, etc.).

  • Each player row shows key performance metrics and rankings within the league.

Interaction Behavior

  • Tapping on Team switches the content to team-level statistics.

  • Tapping on Players switches the content to individual player statistics.

  • Only one tab is active at a time, with a highlighted indicator showing the selected tab.
