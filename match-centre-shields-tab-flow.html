Shields Tab Flow

The Shields tab is part of the Match Centre and displays regional standings for teams grouped into URC regional pools.

Shields Overview

The screen is divided into four regional shield groups:

  • Irish Shield

  • Welsh Shield

  • South African Shield

  • Scottish / Italian Shield

Each group represents teams from that region.

Screen Layout

The Shields screen is presented as a vertically scrollable list of regional sections.

Each regional section contains:

  • Region title (e.g., IRISH, SCOTTISH / ITALIAN)

  • A standings table for teams in that regio

Standings Table Content

For each regional shield, the following information is shown for every team:

  • Team name and logo

  • Matches played (PL)

  • Wins (W)

  • Draws (D)

  • Losses (L)

  • Points difference (PD)

  • Total points (PTS)

Teams are ordered based on their ranking within the regional shield.

User Flow

Match Centre → Shields Tab → Scroll Through Regional Shields → View Regional Team Standings
