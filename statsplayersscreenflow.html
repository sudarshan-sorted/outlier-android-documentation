Players Screen Flow

The Players screen displays individual player statistics across multiple performance categories.

1. Default State

When the Team Stats screen is opened:

  • The Attack category is expanded by default.

  • Other categories are collapsed initially.

  • Each category contains multiple stat leaderboards (e.g. leading points scorer, defenders beaten, etc.).

  • The first player in each leaderboard is shown in a highlighted card layout.

  • Only 5 players are shown in the leaderboard preview.

  • Users must click View All to view the complete list of players.

2. Category Sections

Players statistics are grouped into collapsible categories:

  • Attack

  • Kicking

  • Defence

  • Discipline

  • Lineouts

3. Category Interaction

  • Each category can be expanded or collapsed using the chevron icon.

  • Expanding a category reveals multiple stat subsections inside it.

4. Stat Sub-Sections

Each category contains multiple stat leaderboards (e.g. leading points scorer, defenders beaten, etc.).

5. Leaderboard Display

For each stat subsection:

  • The top-ranked player is displayed in a highlighted large card.

  • Players ranked 2–5 are displayed in a compact vertical list below the highlighted card.

6. View All Action

  • A View All button is shown at the bottom of each stat subsection.

  • Clicking View All opens the full leaderboard list for that stat.

8. User Flow Summary

  1. User opens Stats → Players.

  2. First category is expanded by default.

  3. User scrolls through stat sub-sections.

  4. User expands or collapses categories as needed.

  5. User taps View All to see the complete ranking list.
