Match Centre Shields Tab

The Match Centre Shields tab includes personalized behavior to highlight the user’s selected favorite team within regional shield standings.

Favorite Team Highlighting

  • If the user has selected a favorite team, that team is visually highlighted in the regional shield standings table where it appears.

  • The highlight helps users quickly identify their preferred team’s position within their regional pool.

Highlight Behavior

  • The favorite team row is styled differently from other rows (e.g., background color, accent indicator, etc).

  • The highlight persists across app sessions and updates dynamically when:

    • The user changes their favorite team.

    • Shield standings data is refreshed.

    • The season context is changed.

Non-Personalized Fallback

  • If the user has not selected a favorite team, all teams are displayed with the default table styling.

  • No team is highlighted in this case.
