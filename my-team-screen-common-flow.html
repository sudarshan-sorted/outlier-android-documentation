Common Screen Flow

The My Team section is a personalized area that displays team-specific content for the user’s selected favourite team. This section is only available when the user has chosen a favourite team.

1. Top Bar

  • Left Icon: Back button

    • Navigates back to the previous screen.

  • Center Title: My Team

    • Static title across all My Team sub-screens.

  • Right Action (Season Selector Dropdown)

    • Allows the user to switch the season context (e.g., 2025–26, 2024–25, 2023–24).

2. Access Condition

  • This section is only visible when a favourite team is selected by the user.

  • If no team is selected, this section should not be accessible.

3. Team Overview Card (Persistent Across Sub-Screens)

A team summary card is displayed directly below the top bar and remains consistent across all My Team sub-screens.

Team Card Content

  • Team Logo

  • Team Name

  • Current League Position

  • Last 4 Matches Status

    • Displays recent match outcomes (e.g., W/L indicators).

5. User Flow Summary

  1. User selects a favourite team (prerequisite).

  2. User opens My Team from the drawer or navigation entry point.

  3. Top bar and team overview card are displayed.
