Rounds Tab Flow

The Rounds tab displays fixtures and results grouped by competition rounds and playoffs.

Round Selector

  • A horizontal tab selector is displayed at the top of the screen.

  • Tabs represent regular season rounds and playoff stages.

Regular Season Rounds

  • Round 1 → Round 18

Playoff Rounds

  • Quarter Finals (QF)

  • Semi Finals (SF)

  • Grand Final (GF)

The selector is horizontally scrollable to accommodate all rounds.

Default Behavior

  • When the screen is opened, the current active round (regular season or playoff) is selected by default.

Matches List

  • Matches for the selected round are displayed as vertical match cards.

  • Each card represents a single match.

Match Card States

Each match card can appear in one of the following states:

  • Live Match

    • Displays live score and current match status (e.g., 1st Half, 2nd Half, Full Time).

  • Upcoming Fixture

    • Displays scheduled date and kickoff time.

  • Completed Match

    • Displays final score and match result.

Match Card Interaction

  • Tapping on a match card opens the Match Detail Screen.

User Flow

Match Centre → Rounds Tab → Select Round (R1–R18 / QF / SF / GF) → View Match Cards → Match Detail Screen
